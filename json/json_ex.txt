JSON이란 무엇입니까?
JSON은 약자 JavaScript Object Notation으로 빠르고 쉽게 구문을 분석하고 생성할 수 있는 표준 경량 데이터 교환 형식입니다.

JSON은 XML과 마찬가리로 사람과 컴퓨터 모두 작성하기 쉽고 이해하기 쉬운 텍스트 기반 형식이지만 XML과 달리 JSON 데이터 구조는 XMl 버전보다 대역폭을 덜 차지합니다.
JSON은 두 가지 기본 구조를 기반으로 합니다.

객체: 이것은 키/값 쌍의 모음으로 정의됩니다.(예: key:value)
각 객체는 왼쪽 중괄호로 끝납니다 {}
여러 키/값 쌍은 쉼표로 구분 , 됩니다.

배열: 정렬된 값 목록으로 정의됩니다. 배열은 왼쪽 대괄호로 시작 하여 오른쪽 대괄호로 끝납니다 []
값은 쉼표로 구분 , 됩니다.

값이 될 수 있는 반면 JSON에서, 키는 항상 문자열 string, number, true 또는 false, null 또는 심지어 object나 array 문자열은 큰 따옴표로 묶어야하며 \n, \t와 같은 이스케이프 문자를 포함할 수 있습니다.
